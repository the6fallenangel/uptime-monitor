# uptime-monitor

A concurrent website/API uptime monitoring service written in Go.

Registers URLs to monitor at user-defined intervals, checks them concurrently via a worker pool, and records status/response-time history.

**Work in progress** — built as a hands-on project to learn Go concurrency patterns (goroutines, channels, worker pools) in a genuinely useful context, not a toy exercise.
